================================================================================
                          TROUBLESHOOTING
================================================================================

PROBLEM: "Connection refused"
  Cause: Gateway not running on localhost:8080
  Solution:
    1. Check gateway is running: ps aux | grep gateway
    2. Check port: netstat -tlnp | grep 8080
    3. Test: curl http://localhost:8080/upp/v1/markets?limit=1
    4. Start gateway if needed

PROBLEM: "No markets found"
  Cause: Gateway running but no market data available
  Solution:
    1. Check gateway logs for errors
    2. Verify provider API credentials
    3. Check gateway configuration
    4. Wait for market indexing to complete

PROBLEM: "Timeout"
  Cause: Slow network or gateway overload
  Solution:
    1. Check network connectivity
    2. Reduce batch sizes (--limit flags)
    3. Increase interval (--interval flags)
    4. Check gateway logs for performance

PROBLEM: "Invalid JSON"
  Cause: Gateway returning malformed response
  Solution:
    1. Verify gateway version matches examples
    2. Check market data format in gateway
    3. Test directly: curl http://localhost:8080/...

PROBLEM: "Script won't run"
  Cause: Python version or syntax issue
  Solution:
    1. Check Python version: python3 --version (need 3.9+)
    2. Verify syntax: python3 -m py_compile script.py
    3. Check imports: python3 -c "import script"
    4. Run with verbose: python3 -u script.py
