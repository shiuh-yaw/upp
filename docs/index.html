Quick Integration

TypeScript
import { UPPClient } from '@upp/sdk';

const client = new UPPClient({
  baseURL: 'http://localhost:8080',
  apiKey: process.env.UPP_API_KEY
});

// Fetch markets across all providers
const markets = await client.markets.list({
  query: 'Trump 2024',
  limit: 10
});

for (const market of markets) {
  console.log(market.name);
}
Python
from upp import UPPClient

client = UPPClient(
    base_url="http://localhost:8080",
    api_key="your-api-key"
)

# Fetch markets across all providers
markets = client.markets.list(
    query="Trump 2024",
    limit=10
)

for market in markets:
    print(market.name)

Quick Start

1

Install SDK

Add the UPP SDK to your project:

npm install @upp/sdk
2

Create Client

Initialize the client with your configuration:

const client = new UPPClient({...})
3

Fetch Markets

Start querying markets and placing orders:

await client.markets.list()
